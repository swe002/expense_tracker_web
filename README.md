# Expense Tracker Web App

A modern Expense Tracker web application built using Flask and SQLite.

## Features
- User Registration & Login
- Animated Login Page
- Dark Mode / Light Mode
- Dashboard with Charts
- Expense Category Analysis
- Theme Switcher
- Secure Authentication

## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS, JavaScript

## How to Run
1. Install Flask
2. Run: python app.py
3. Open browser at http://127.0.0.1:5000/
